Result file validation

Previous | Next

Result file validation is a tool(optional) that validates MzIdentML file faster. This validation is currently only supported for complete submissions that contains results file as MzIdentML.

Validations run by Result file validation

  • XML Schema validations
  • Spectra references
  • Check Metadata
  • Check if any missing spectra presence
  • Check if protein, peptides, PSMs are presence
  • Recalculate Delta Mass randomly
    •

    Please note that Result file validation will check only the smalled MzIdentML file in the complete submission and it will do a basic validation to help preventing from submitting inaccurate submission files to the PRIDE. A more complete validation will be done, once you submit data.
